# Router-on-a-Stick Project



## 📖 Overview
This project demonstrates **Inter-VLAN Routing (Router-on-a-Stick)** using a single router interface.

---

## 🧠 Key Concepts
- VLANs (Virtual LANs)
- 802.1Q Trunking
- Subinterfaces
- DHCP Configuration
- Inter-VLAN Routing
- Network Troubleshooting
